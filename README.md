## 📖 Sobre o Projeto

O **AncestryLens** é uma ferramenta web que utiliza **inteligência artificial e visão computacional** para estimar a similaridade facial entre duas pessoas. Basta enviar duas fotos e o sistema realiza uma análise biométrica comparativa, gerando um percentual de semelhança e um relatório visual detalhado.

### ✨ Funcionalidades

- 📸 **Upload de duas imagens** — Arraste ou clique para enviar as fotos dos Sujeitos A e B
- 🤖 **Detecção facial com IA** — Reconhecimento automático de rostos usando redes neurais (SSD MobileNet + Face Landmark 68)
- 📊 **Análise por regiões** — Comparação detalhada de sobrancelhas, olhos, sorriso e mandíbula
- 🧬 **Modo Genético** — Algoritmo ajustado para comparações entre pessoas de idades diferentes (pai/filho)
- 🎨 **Malha biométrica visual** — Visualização tech dos pontos faciais detectados
- 📥 **Download do resultado** — Gere um certificado visual em formato de imagem para compartilhar
- 🔒 **Privacidade total** — Todas as imagens são processadas **localmente no navegador**, sem envio a servidores

## 🚀 Como Usar

1. Acesse **[stehlima.github.io/parentesco](https://stehlima.github.io/parentesco/)**
2. Envie a foto do **Sujeito A** (clique ou arraste)
3. Envie a foto do **Sujeito B**
4. Aguarde a detecção facial (✅ aparecerá quando o rosto for encontrado)
5. Clique em **INICIAR ESCANEAMENTO**
6. Veja o resultado com o percentual de semelhança!
7. Ative o **Modo Genético** para comparar pessoas com diferença de idade
8. Baixe o resultado como imagem clicando em **BAIXAR RESULTADO**
